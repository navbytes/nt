nt — TUI design mockup

A static HTML preview of the planned Bubble Tea TUI (SPEC §12), for design review before any Go is written. Nothing here is interactive — it shows the visual language: layouts, colors, status glyphs, grouping, the detail overlay with links/backlinks, and the keybar. Colors approximate a Tokyo-Night terminal theme rendered with Lipgloss.
